# Mi Taller Dev

Primer proyecto frontend de mi portafolio, creado con HTML, CSS y JavaScript.

## Descripción

Este proyecto es una página personal inicial donde presento mi ruta como desarrollador frontend junior.  
La página resume mi enfoque de aprendizaje, tecnologías iniciales y proyectos que voy a construir para mi portafolio.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Git
- GitHub
- Live Server

## Secciones del proyecto

- Presentación personal
- Enfoque de aprendizaje
- Proyectos futuros
- Stack inicial

## Proyectos que voy a construir

- Calculadora de Reventa
- Dashboard de Inventario
- DealTracker MX

## Objetivo

Usar este proyecto como punto de partida para construir un portafolio enfocado en desarrollo frontend, dashboards, inventarios, automatización y análisis de oportunidades de negocio.
